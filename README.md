# FilmAI — AI-киносоветчик

Готовая первая версия сайта без регистрации. Она работает сразу в демо-режиме и умеет:

- подбирать пять фильмов по свободному запросу;
- понимать настроение, жанры и упомянутые любимые фильмы;
- объяснять рекомендации без спойлеров;
- сохранять избранное в браузере через `localStorage`;
- открывать поиск трейлера;
- подключать OpenAI для настоящих AI-рекомендаций;
- подключать TMDB для рейтингов, описаний и постеров.

## Быстрый запуск

Нужен Node.js 18 или новее.

```bash
cd filmai-advisor
npm start
```

Откройте в браузере:

```text
http://localhost:3000
```

Для разработки с автоматическим перезапуском:

```bash
npm run dev
```

## Подключение настоящего AI

1. Скопируйте `.env.example` в `.env`.
2. Вставьте серверный ключ OpenAI:

```env
OPENAI_API_KEY=ваш_ключ
OPENAI_MODEL=gpt-5.6
```

Ключ хранится на сервере и никогда не передаётся в браузер.

## Подключение TMDB

Добавьте API Read Access Token:

```env
TMDB_READ_TOKEN=ваш_bearer_token
```

Либо обычный API key:

```env
TMDB_API_KEY=ваш_api_key
```

После перезапуска сервера AI-рекомендации будут дополняться данными TMDB и постерами.

- ограничьте расходы OpenAI в панели проекта;
- замените простое ограничение запросов на Redis/Upstash при нескольких серверах;
- добавьте собственный домен и HTTPS;
- ознакомьтесь с требованиями атрибуции и использования данных TMDB;
- добавьте политику конфиденциальности, если подключите аналитику или рекламу.
